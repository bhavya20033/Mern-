# EventHub - Static Version

This is a static HTML/CSS/JS version of the EventHub Mini Event Platform.
It was converted from a MERN stack application to run on GitHub Pages.

## Features
- **Event Listing**: View upcoming events.
- **Event Details**: View details of a specific event.
- **RSVP**: Join events (simulated with LocalStorage).
- **Authentication**: Login/Signup (simulated with LocalStorage).
- **Create Event**: Add new events (simulated with LocalStorage).

## Architecture
- **HTML5**: Semantic structure.
- **CSS3**: Responsive design with CSS variables.
- **JavaScript (ES6+)**: Logic for routing, state management, and DOM manipulation.
- **LocalStorage**: Persists user data and events across sessions.

## MERN Archive
The original MERN (MongoDB, Express, React, Node.js) source code is archived in the `mern-archive/` directory for reference.
